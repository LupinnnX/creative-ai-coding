# NOVA Framework v6.0

> *"Activate with keywords. Ship legendary work."*

## Overview

NOVA (Network of Virtual Agents) is a world-class multi-agent coordination system with 6 specialized AI agents working in parallel on complex development tasks. Each agent is a sovereign specialist with distinct expertise, cognitive patterns, and activation keywords.

## Quick Start

1. **Activate an agent**: Say "Activate POLARIS" (or any agent name)
2. **Load steering**: Read the relevant steering file for your workflow
3. **Follow the Loop**: Every task goes through the Cognitive Loop v2.0
4. **Hand off properly**: Use structured handoffs between agents

## The Cognitive Loop v2.0

Every significant task follows this enhanced cognitive cycle:

```
0. META-PLAN     → "How should I approach this?"
1. DRAFT         → Generate initial solution (fast)
2. SELF-CORRECT  → "What mistakes did I just make?"
3. CRITIQUE      → "What would a Principal Engineer reject?"
4. REFINE        → Address each critique systematically
5. VERIFY        → Prove it works (run code, check docs)
6. CHECKPOINT    → Record reasoning for future learning
```

**Rule**: No output leaves until it survives the Loop.

## Workflow Patterns

The framework includes 7 proven coordination patterns:

1. **Sequential Chain**: Linear dependencies (A → B → C)
2. **Parallel Fan-Out/Fan-In**: Independent subtasks running simultaneously
3. **Supervisor-Worker**: POLARIS oversees multiple workers
4. **Intelligent Routing**: Match tasks to best-fit agents
5. **Contract-First Pair**: Frontend + Backend with shared API contract
6. **Gate-Keeper**: Mandatory checkpoints (SIRIUS, ARCTURUS)
7. **Self-Improving Loop**: Learn from task outcomes

## Handoff Protocol

When transitioning between agents, always include:

```
HANDOFF: [FROM_AGENT] → [TO_AGENT]

WHAT I DID:
- [Completed work summary]

WHAT YOU NEED TO DO:
- [Clear, actionable next steps]

CONTEXT:
- [Key files, decisions, constraints]

WATCH OUT FOR:
- [Known risks, edge cases, blockers]
```
